# ZaeUI_FriendlyPlates

Show friendly player nameplates in name-only mode with class colors and custom font size — works in dungeons, raids and open world, including on Blizzard's "forbidden" nameplates in instances.

Works with **Midnight (12.0.0+)**.

## How it works

- **Friendly Nameplates**: Activates friendly player nameplates via native CVars
- **Name Only Mode**: Hides health bars, cast bars and classification icons — shows only the player name
- **Class Colors**: Colors each player name by their class (Warrior brown, Paladin pink, etc.)
- **Custom Font Size**: Modifies global nameplate font objects to change the text size (works even on forbidden nameplates in instances)
- **Options Panel**: Full UI configuration via Escape → Options → AddOns → ZaeUI → FriendlyPlates

## Commands

| Command | Description |
|---------|-------------|
| `/zfp` | Open the options panel |
| `/zfp toggle` | Toggle friendly nameplates on/off |
| `/zfp size 14` | Set font size (8 - 28) |
| `/zfp reset` | Reset all settings to defaults |
| `/zfp help` | Show help |

## Features

- Friendly player nameplates with name-only display
- Class-colored player names for quick identification
- Custom font size (8–28) via global font objects
- Works in dungeons and raids (forbidden nameplates supported)
- Settings persist across sessions
- Compatible with Midnight's nameplate overhaul
- Requires [ZaeUI_Shared](../ZaeUI_Shared/)
